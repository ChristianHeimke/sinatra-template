
# Sinatra Template

A simple template for sinatra projects

## Environment
* Ruby 2.6.5
* Sinatra 2.0.5
* MDBootstrap
* slim


## Development
1. Bundle gems locally:

```
$ bundle install
```

2. Run app from root. Will load app on http://localhost:4567 by default.

```
$ bundle exec rackup config.ru -p 4567
```
